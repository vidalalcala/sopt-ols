\subsection{Detailed Description}
Gradient sampler for the quadratic loss in Le\-Cun , Schaul, Zhang (2013). 

Gradient sampler for the loss function \[ L = 0.5 x H x^T \]

where $ x = \alpha - (\alpha_{*} + \Sigma Z) $ and $ Z $ is a standard normal random variable. Vectors like $ \alpha,Z $ are row vectors and the matrices $ H,\Sigma $ are square matrices.\-The default parameters $ H,\Sigma,\alpha_{*} $ are two identity matrices and the zero vector.
